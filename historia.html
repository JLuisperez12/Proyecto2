HISTORIA DE JAVASCRIPT

Años 1990: Orígenes y Desarrollo Temprano

  • 1995: Brendan Eich desarrolla JavaScript en Netscape en tan solo 10 días, inicialmente llamado "Mocha".
  • 1996: Netscape lanza JavaScript como "LiveScript" y luego cambia el nombre a "JavaScript".
  • 1997: JavaScript se estandariza bajo la especificación ECMAScript por ECMA International.

Años 2000 - 2010: Expansión y Evolución

  • 2005: Se lanza AJAX, que permite la creación de aplicaciones web interactivas sin recargar la página.
  • 2008: Google lanza Chrome, acelerando el uso de JavaScript en navegadores.
  • 2009: Node.js es creado por Ryan Dahl, permitiendo usar JavaScript en el servidor.

Años 2010 - Presente: Dominio y Modernización

  • 2015: Se lanza ECMAScript 6 (ES6), una actualización importante que introduce clases, promesas y otras mejoras.
  • 2018: JavaScript sigue siendo el lenguaje dominante para el desarrollo web, con nuevos frameworks como React y Vue.js.
  • 2020: JavaScript se utiliza ampliamente en aplicaciones móviles, de servidor y de escritorio gracias a tecnologías como Electron.
